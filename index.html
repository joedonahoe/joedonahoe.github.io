2015.09.14 Let's Get This Going

I decided the other day that I needed a place where I can put the kind-of-useful stuff I build as I'm learning new things and practicing things I think I know. I tried WordPress, but I didn't really want a blog. I just wanted a chronological list (this page) and a portfolio (the Stuff page).

So this is that. More to come.

Some design notes:

  • Since I like simplicity, control, and full understanding, the HTML here is completely manual and edited in TextPad.
  • I un-div'd this page from an earlier document to clean it up.
  • I'm going with cornflower blue because it looks okay with the default HTML link color. I prefer to use pre-built constants instead of hunting for the perfect hexadecimal code that I'll need to document anyway.
  • I learned that I'm not supposed to put an <ul> or an <ol> inside of a <p>. Stack Overflow to the rescue.
  • I also prefix my working files with the date in YYYY.MM.DD format. As in "2015.08.15 Research That Weird Sales Order.sql". So I'm using the same format here.

On that last point, I am writing with the expectation that I am 95% of my own audience. And 2% of the audience is seeing how I work. The other 3% is probably my mother.
